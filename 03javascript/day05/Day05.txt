数组
1、数组是引用类型的对象
	引用类型的数据时保存在堆(heap)中的,在栈(stack)会对堆中的地址进行引用


	将数组作为参数传递给函数的时候，实际上是按"引用传递"的 - 传递的是数组的地址，而不是数组中的值

	null 与 undefined
	undefined : 声明变量未赋值
	null : 让对象不指向任意空间，即主动释放空间
2、关联数组
	php中：$array=["name"=>"WH.L"];
	以字符串做为元素下标的数组，就是关联数组
	以数字作为元素下标的数组，就是索引数组

	在JS中的关联数组:
		var array = [];
		array["字符串下标"]=值;
		ex:
			array["西游记"] = "古代神话故事";
			array["红楼梦"] = ["贾宝玉","林黛玉","刘姥姥"];

		注意：
			1、关联数组中，字符串下标的内容是不记录到length中的
			2、只能通过 字符串 做下标取值
3、for ... in
	遍历出任意数组中的字符串下标 以及 索引下标
	语法：
		for(var 变量 in 数组){
			//变量：字符串下标 或 索引下标
		}
4、冒泡排序
	1、什么是冒泡排序
		排序算法之一，将一组数字进行排序,小的数字向前排，大的数字向后排
			var arr=[23,9,78,6,45]
			两两比较，小的靠前
		特点：
			1、n个数字，则比较 n-1轮(外层循环)
				for(var i=1;i<arr.length;i++)
			2、轮数增加，比较次数减少(内层循环)
				for(var j=0;j<arr.length-i;j++){
					if(arr[j]>arr[j+1]){
						//交换 arr[j] 和 arr[j+1]的位置
					}
				}
